# 🌸 Happy 19th Birthday - Quinter Aiz 💕

A beautiful, romantic, floral & elegant birthday website dedicated to **Quinter Aiz** on her 19th birthday. Built with pure HTML & CSS — no JavaScript required!

---

## ✨ Features

### 🎨 Design & Theme
- **Pastel pink, lavender, and gold** color palette
- **Floating rose petals** animation across the screen
- **Elegant script fonts** (Great Vibes & Playfair Display)
- **Dark mode toggle** (🌙 button, top-left corner)
- **Sticky navigation bar** for easy browsing

### 🎵 Music
- **"Perfect" by Ed Sheeran** — click the 🎵 button (top-right) or the "Play Our Song" button to open on YouTube
- Dedicated **"Our Song"** section with lyrics quote

### 💕 Interactive Elements
- **Tabbed Sections** — Switch between Feelings, Wishes, Memories & Dreams
- **Flip Cards** — Hover to reveal surprise messages
- **Secret Love Letter** — Click to expand a hidden love letter
- **19 Reasons I Love You** — Expandable accordion with all 19 reasons
- **Love Compatibility Quiz** — Answer 2 questions to reveal "soulmate" result
- **Star Rating** — Rate your love story (with custom responses)
- **Send Love Heart** — Tap the heart for an animated love message
- **Confetti Burst** — Pop the confetti button for celebration emojis
- **Birthday Wishes Wall** — Tilted cards with rotating wishes

### 💍 Proposal Section
- **Glowing golden ring** with pulse animation
- **"Yes, Forever!" button** — Links directly to WhatsApp DM with a pre-filled romantic message
- Dark romantic background with rose petals

### 📸 Photo Gallery
- 6 placeholder slots ready for your photos
- Hover overlay with captions
- Zoom-on-hover effect

### 📍 Love Story Timeline
- Visual timeline from "The Beginning" to "Forever & Beyond"
- Hover animation on each milestone

### 🤝 Promises Section
- Hoverable promise cards with gold accents

## 🚀 How to Deploy (Get a Shareable Link)

### Option 1: Netlify Drop (Easiest — 30 seconds)
1. Save the HTML code as `index.html`
2. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
3. Drag and drop your file
4. Get your free link instantly! ✅

### Option 2: GitHub Pages (Free & Permanent)
1. Create a GitHub account at [github.com](https://github.com)
2. Create a new repository (e.g., `happy-birthday-quinter`)
3. Upload the HTML file as `index.html`
4. Go to **Settings → Pages → Deploy from main branch**
5. Your link: `https://yourusername.github.io/happy-birthday-quinter/`

### Option 3: Tiiny.host (Simple Upload)
1. Go to [tiiny.host](https://tiiny.host)
2. Upload your `index.html`
3. Get a shareable link instantly

## 📝 How to Customize

### Add Your Photos
Replace the gallery placeholder `<div>` elements with actual images:
```html
<div class="gallery-item">
  <img src="your-photo.jpg" style="width:100%; height:100%; object-fit:cover;">
  <div class="overlay">Your caption here 💕</div>
</div>
```

### Change the WhatsApp Link
Find this line in the proposal section and replace with your link:
```html
<a href="https://wa.me/message/MLSPI6W5HFDBG1" target="_blank" ...>
```

### Edit Messages
All text content is in plain HTML — search for the text you want to change and update it directly.

### Change the Song
Replace the YouTube URL in both locations:
```
https://www.youtube.com/watch?v=2Vv-BfVoq4g
```
